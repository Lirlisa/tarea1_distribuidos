La tarea se encuentra dentro del directorio 'tarea1_distribuidos', la asignación de funciones
para cada maquina es la siguiente:
-Máquina 1: Cliente
-Máquina 2: Logística
-Máquina 3: Finanzas
-Máquina 4: Camiones

Una vez dentro de 'tarea1_distribuidos' se debe ingresar al directorio de mismo nombre que la
función de la maquina, pero sin tildes ni mayusculas, ej.:
    Máquina 1 -> cliente

Luego basta con hacer make y se ejecutará automáticamente.
Se recomienda iniciar logistica antes que cualquier otra máquina.
